# My Pakistan Travel Diary - Website

This is the exported travel website from the desktop application.

**View Online:** https://YOUR_USERNAME.github.io/MyPakistanDiary/

## Features
- Multi-page tourism website
- Interactive maps
- YouTube travel videos
- Newspaper and magazine features
- Real travel photos
- Light/Dark mode toggle

## Cities Covered
- Islamabad
- Lahore
- Taxila
- Murree
- Rawalpindi

## To Deploy to GitHub Pages:

1. Create a new repository: `MyPakistanDiary-Website`
2. Copy the `export/` folder contents to the repository
3. Go to Settings → Pages → Deploy from a branch → main
4. Your site will be live at: `https://YOUR_USERNAME.github.io/MyPakistanDiary-Website/`
